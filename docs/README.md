# Project Documents

This folder keeps non-runtime project documents out of the repository root.

- `archive/2026-03-audits/` contains older audit reports, implementation summaries, setup notes and planning snapshots from the March 2026 cleanup phase.
- `archive/2026-04-root-artifacts/` contains historical root-level generated files and one-off snippets kept only for traceability.
- `archive/2026-04-stale-scripts/` contains old duplicate helper scripts removed from active tool paths.
- The root stays reserved for the active entry points: `README.md`, `AGENTS.md`, `CLAUDE.md`, `TECHNICAL_DEBT_BACKLOG.md`, `llms.txt` and runtime files.

When adding a new report, keep it here unless it is an active operational file needed by agents or deployment.
