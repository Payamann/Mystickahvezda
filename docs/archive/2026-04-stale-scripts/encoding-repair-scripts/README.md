# Encoding Repair Scripts

These scripts were one-off repair helpers for older mojibake, content, SEO,
calendar, horoscope, font path, CSS, and icon fixes. They intentionally include
broken character samples as replacement inputs.

Active encoding prevention now lives in:

- `server/scripts/check-encoding.py`
- `npm run check:encoding`
- `npm run test:verify`

Keep these files archived for traceability; do not treat them as current
maintenance tooling.
