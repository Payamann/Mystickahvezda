# Bulk HTML Mutators Archive

This folder keeps old one-off scripts that made broad HTML, CSS, SEO,
performance, or image asset edits across the workspace.

They are archived because they were not referenced by active npm scripts, some
used hard-coded local paths, and several predate the current CSP/build/audit
workflow or generated inline CSS/event handlers. Do not run them as current
maintenance tooling without reviewing and modernizing the logic first.
