Manual Activation (Development Only)

Pouzij tento endpoint pro testovani premium funkci bez Stripe:

POST http://localhost:3001/payment/subscription/activate-manual
Authorization: Bearer YOUR_TOKEN

Body:
{
  "userId": "YOUR_USER_ID",
  "planType": "premium_monthly",
  "durationDays": 30
}
